# Documentação Nexo

Nexo é um **framework** frontend declarativo e AI-native para landing pages e microsystems. Tudo gira em torno de um `schema.json`, pastas de componentes com Shadow DOM e export estático sem build step pesado.

## Por onde começar

| [Guia de uso](./guia-de-uso.md)     | Tutorial prático: instalar, criar projeto, componentes, dev e build  |
| [Especificação](./DOCUMENTATION.MD) | Referência completa (schema, props, assets, HMR, export, tipos, CLI) |

## Comandos rápidos

```bash
nexo init meu-site
cd meu-site
nexo dev
nexo build
```
